# 垃圾分类查询百科

一个纯前端的垃圾分类查询与分类指南工具，帮助用户快速了解各类垃圾的正确分类方式和投放要求。

## 技术栈

- Vite 5
- React 18
- TypeScript
- 纯前端实现，无后端依赖
- 数据存储于浏览器 localStorage

## 启动方式

```bash
npm install
npm run dev
```

启动后访问 http://localhost:5181。可通过 `VITE_PORT` 覆盖默认端口。

## 功能说明

### 1. 实时搜索
在首页搜索框输入物品名称（如"旧报纸"、"电池"、"西瓜皮"），下方实时显示匹配结果，每条结果用颜色标记分类归属：
- 🔵 蓝色 — 可回收物
- 🔴 红色 — 有害垃圾
- 🟢 绿色 — 厨余垃圾
- ⚫ 灰色 — 其他垃圾

### 2. 四大分类入口
首页展示四大垃圾分类快捷入口卡片，点击进入某分类查看该类别下的全部物品列表和正确投放方式。

### 3. 物品详情
点击任意物品查看完整信息：
- 分类归属说明
- 正确投放要求
- 常见错误提醒

### 4. 拍照识物（模拟）
上传一张垃圾物品照片，系统基于文件名关键词匹配模拟识别，给出最可能的分类结果和置信度。

### 5. 分类指南
图文卡片形式介绍四大垃圾类别的定义、颜色标识、包含物品举例和投放注意事项。

### 6. 主题切换
支持浅色和深色两种主题模式，切换后自动保存到 localStorage。

### 7. 搜索历史
自动记录最近 20 条搜索查询，支持一键清空。

## 数据规模

内置 310+ 种常见垃圾物品数据，四类分布均匀，每种物品包含：
- 分类归属
- 投放说明
- 常见错误提示
- 搜索关键词
